## 🚀 TRANSICIÓN A EXCEL

Cuando tengas 2-3 semanas de datos:

1. **Crear columnas en Excel:**
   - Fecha
   - Descripción
   - Monto
   - Tipo (Ingreso/Gasto)
   - Categoría
   - Fuente

2. **Importar datos del cuaderno**

3. **Crear fórmulas simples:**
   - `=SUMIF(C:C,"<0",C:C)` para gastos totales
   - `=SUMIF(C:C,">0",C:C)` para ingresos totales
   - Tablas dinámicas para análisis por categoría

4. **Validar que todo cuadre**

---

## 🔄 INTEGRACIÓN CON WINM

Una vez que tengas datos organizados en Excel:

1. **Exportar a CSV** desde Excel
2. **Usar scripts de WINM** para importar
3. **Configurar reglas de auto-categorización** basadas en tus patrones
4. **Verificar dashboard** contra tus datos manuales
